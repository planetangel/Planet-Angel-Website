Backup

This page is used to backup your SEF Advance settings data.
You can export your settings and download the backup file to your computer. The file will contain all of your configuration settings and aliases. This backup file can be later imported to restore all the previous data. This is convenient when you need to reinstall SEF Advance or upgrade to the latest version. The backup file from a previous version of SEF Advance can also be used in newer versions to import your previous data.

Toolbar Actions

Import - Imports the settings from the SEF Advance backup file.

Export - Exports the settings to a SEF Advance backup file for download.

Cancel - Cancels the backup and returns to the control panel.

Field Details

Backup file - Choose the local backup file for import.
