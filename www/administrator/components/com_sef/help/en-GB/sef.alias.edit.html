New / Edit Alias

This page is used to either add a new Alias or edit an existing one.

Toolbar Actions

Save - Saves the Alias pair and returns to Alias list.

Cancel - Cancels the modifications and returns to Alias list.

Field Details

Non-SEF URL - This is the relative link of the URL that you want aliased.
Enter it with no leading slash (begins with index.php?option=...). To find out particular relative URL temporarily disable SEF in Global Configuration.
You can also copy-paste the original SEF URL from the address bar, it will get translated to non-sef URL automatically for you.

Alias - Alias that will be used instead of automatically generated URL.
Enter it without any leading or trailing slash or suffix.

Title - Custom page title that will override the automatically generated one for that page.

Description - Meta description entry to be used instead of general one.

Keywords - Meta keywords that will override the globally defined keywords, separate by comma.

Canonical - Custom canonical tag to insert in page head section instead of automatically generated one.

Counters - The counters will display how many characters you have left for each Metadata entry based on recommended maximal values.
