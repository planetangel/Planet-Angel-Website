Alias & Meta List

This page shows a list of the Aliases that have been saved in the database.
If you are not happy with the look of a particular URL you can manually enter an alias for that link here. This alias, if published, will then be used instead of automatically generated SEF URL.

Along with the alias you can add custom Meta data for that particular page including Title, Description and Keywords.
This will override automatically generated Meta data and give you greater SEO control over each individual page.

You may enter text in the "Filter" box to refine the list based on whether the searched string is found in the Alias. Type in any string and press Enter.
You may also limit the list by selecting the number of presented results from the dropdown box. You can move through the pages by clicking on links in the navigation bar on the bottom of the list.

Toolbar Actions

You "select" items by clicking the checkbox at the start of each row.

New - Click the 'New' icon to create a new Alias.

Edit - Click the 'Edit' icon to edit the Alias entry. Clicking the linked URL will also perform this action.

Publish and Unpublish - Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected Aliases.

Delete - Click the 'Delete' icon to delete selected Alias entries. Note that they are then completely removed from the database and can't be restored.

Cancel - Cancels the alias list and returns to the control panel.

Column descriptions

Non-SEF URL - This is the relative link of the URL that you want aliased.

Alias - Alias that will be used instead of automatically generated URL.

Published - This shows whether the alias is published. A click on the publish icon will toggle the publishing state of the alias.
