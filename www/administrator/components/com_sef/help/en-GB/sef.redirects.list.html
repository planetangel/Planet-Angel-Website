Redirect List

This page shows a list of the custom redirects saved in the database.
Here you can define redirects of certain URLs to another ones. It can be an old link that changed or even a short version of existing document. When user requests the source URL he will be redirected to the target of your choice and selected header will be sent to browser.

You may enter text in the "Filter" box to refine the list based on whether the searched string is found in the Source URL. Type in any string and press Enter.
You may also limit the list by selecting the number of presented results from the dropdown box. You can move through the pages by clicking on links in the navigation bar on the bottom of the list.

Toolbar Actions

You "select" items by clicking the checkbox at the start of each row.

New - Click the 'New' icon to create a new Redirect.

Edit - Click the 'Edit' icon to edit the Redirect entry. Clicking the linked URL will also perform this action.

Publish and Unpublish - Click the 'Publish' or 'Unpublish' icon to publish or unpublish any number of selected Redirects.

Delete - Click the 'Delete' icon to delete selected Redirect entries. Note that they are then completely removed from the database and can't be restored.

Cancel - Cancels the Redirects list and returns to the control panel.

Column descriptions

Source URL - This is the relative link of the URL that you want redirected.

Target URL - The address the user will be redirected to.

Type - The status code to be sent to browser.

Published - This shows whether the Redirect is published. A click on the publish icon will toggle the publishing state of the Redirect.
