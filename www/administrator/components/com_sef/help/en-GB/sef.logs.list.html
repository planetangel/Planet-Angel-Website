Log List

This page shows a list of the logged URLs that resulted in Not Found error and have been saved in the database.

SEF Advance will log requested URLs which were not found, along with the time, IP address and referer of the click, so you can fix the broken links or instruct the referers.
Note that logging must be turned ON in configurations before SEF Advance can log clicked URLs.

You may enter text in the "Filter" box to refine the list based on whether the searched string is found in the URL. Type in any string and press Enter.
You may also limit the list by selecting the number of presented results from the dropdown box. You can move through the pages by clicking on links in the navigation bar on the bottom of the list.

Toolbar Actions

You "select" item by clicking the checkbox at the start of each row.

View - Click the 'View' icon to show more information on the checked URL entry. Clicking the linked URL will also perform this action.

Delete - Click the 'Delete' icon to delete selected URL entries. Note that they are then completely removed from the database and can't be restored.

Empty - Click the 'Empty' icon to delete all logged entries and empty the database. Note that all entries are then completely removed from the database and can't be restored.

Cancel - Cancels the log list and returns to the control panel.

Column descriptions

URL - The relative link of the URL that resulted in Not Found error.

Time - Time when the URL was accessed.

IP - IP address from where the URL was accessed.

Referer - The site on which the URL link was clicked.
