New / Edit Redirect

This page is used to either add a new Redirect or edit an existing one.

Toolbar Actions

Save - Saves the Redirect and returns to Redirect list.

Cancel - Cancels the modifications and returns to Redirect list.

Field Details

Source URL - This is the relative link of the URL that you want redirected.
Enter it with no leading slash (example: some/url/).

Target URL - The address that the user will be redirected to.
Enter it with no leading slash (example: animals/pets/cat/) or full URL for extarnal redirect (example: http://www.domain.com).

Redirect type - The status code to be sent to the browser.
